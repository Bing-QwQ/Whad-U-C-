# 简介
通过C++来进行文件输入输出操作以及对char*类型数据操作的练习示例（以操作网易云音乐的音乐播放缓存文件为例）
## ver 0.0.1
-对缓存文件（.uc文件）进行基本的文件读取，文件输出
-识别文件后缀，否则不执行
### 使用说明
-编译“Whad U C.cpp”即可
